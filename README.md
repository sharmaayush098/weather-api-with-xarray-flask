# Flask Xarray Weather API

This project demonstrates how to use Xarray with geospatial coordinates
to build a weather analytics API using Flask.

## Features

- Query temperature by latitude and longitude
- Time-series temperature analysis
- Regional temperature averages
- Monthly aggregation
- Swagger API documentation

## Tech Stack

- Flask
- Xarray
- NetCDF
- Dask
- Swagger

# Installation Stepp

1. Create python virtual env.
2. Run cmd : pip install -r requirement.txt
3. python app.py
